Detailed Description

Notifications API Controller All endpoints require authentication - users can only access their own notifications GET /api/notifications - Paginated list of user's notifications GET /api/notifications/unread-count - Unread notification count POST /api/notifications/{id}/mark-read - Mark a single notification as read POST /api/notifications/mark-all-read - Mark all notifications as read
